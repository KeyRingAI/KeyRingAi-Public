# Attachments Workflow Example

This example shows how to use attachments without over-sharing data.

## Goal

Ask two providers to summarize a short public document excerpt.

## Setup

Create or choose a non-sensitive text file. Open Attachments and add the file. Scope it only to the providers that need it. Choose first-lines or raw mode depending on the size and purpose of the file.

## Prompt

Ask for a concise summary, key risks, and open questions. State what the file represents so the model does not have to infer context from the attachment alone.

## Review

Compare provider output. Check whether each model used the attachment accurately. Remove the attachment when the task is complete if it is no longer needed.

## Public Boundary

This example avoids private files, credentials, regulated data, customer data, and proprietary source code.
