# Roundtable Workflow Example

This example shows a public-safe way to run a multi-provider Roundtable evaluation.

## Goal

Compare how three providers reason about the same non-sensitive planning question.

## Setup

Configure at least three provider/model participants. Verify each participant with a short direct prompt before starting Roundtable. Use a public or synthetic topic such as: "Compare three approaches for organizing a small documentation launch."

## Run

Open Roundtable. Choose participants and models. Select a discussion mode such as panel, debate, or collaborative. Set a modest turn limit. Confirm that no sensitive attachments are active. Start the session and monitor participant status.

## Review

Read the transcript for agreement, disagreement, assumptions, missing risks, and practical next steps. Do not treat consensus as proof. Export the transcript only after checking that it contains no private prompts or sensitive data.

## Public Boundary

This example uses synthetic content and does not include proprietary source code, customer data, or internal prompts.
