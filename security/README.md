# Security

Please report suspected security issues privately.

Do not open public GitHub issues for suspected vulnerabilities, security bugs, credential exposure, abuse paths, or privacy concerns.

## Private Reporting Contact

Security contact: security@[your-domain].com

Please replace this placeholder with the correct monitored security contact before making the repository public.

## What To Include

When reporting a security issue, include as much detail as possible:

- A clear description of the issue
- Steps to reproduce, if safe to share
- Affected product area or public resource
- Potential impact
- Any relevant logs, screenshots, or proof of concept details
- Your preferred contact information for follow-up

## Scope

This public repository contains documentation and public developer resources only. The proprietary KeyRing AI application source code is not included here.

If you discover secrets, private configuration, customer data, or proprietary source code in this public repository, report it privately immediately.
